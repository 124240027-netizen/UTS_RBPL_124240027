📦 INSTALLATION STEPS
====================

1. CREATE DATABASE:
   - Open phpMyAdmin: http://localhost/phpmyadmin
   - Create database: rantaugrafi_db
   - Import: database.sql

2. VERIFY CONFIGURATION:
   - Check: includes/config.php
   - Ensure DB credentials are correct
   - Verify uploads folder permissions (chmod 777)

3. START SERVER:
   - Start Apache & MySQL in XAMPP Control Panel
   - Open browser

4. ACCESS WEBSITE:
   - Homepage: http://localhost/Rantaugrafi/
   - Admin: http://localhost/Rantaugrafi/admin/dashboard.php
   - Quick Start: http://localhost/Rantaugrafi/quickstart.php

5. LOGIN WITH DEFAULT ADMIN:
   - Email: admin@rantaugrafi.com
   - Password: password123
   - ⚠️ Change password after first login!

📌 IMPORTANT NOTES
==================

1. DATABASE IMPORT:
   - Use phpMyAdmin Import tab
   - Or run: mysql -u root -p rantaugrafi_db < database.sql
   - Verify all 8 tables are created

2. FILE PERMISSIONS:
   - uploads/ folder should be writable
   - chmod 777 uploads/ (Linux/Mac)
   - Right-click > Properties > Security > Edit (Windows)

3. CONFIGURATION:
   - Edit includes/config.php if using different DB credentials
   - Set correct SITE_URL if using different port/domain

4. SECURITY:
