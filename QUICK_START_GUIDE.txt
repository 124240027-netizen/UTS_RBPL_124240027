⏱️ SETUP HANYA 5 MENIT!
=======================

STEP 1: IMPORT DATABASE (2 menit)
---------------------------------
1. Buka browser → http://localhost/phpmyadmin
2. Login dengan username: root (kosongkan password)
3. Klik "New" atau "Create new database"
4. Nama: rantaugrafi_db
5. Charset: utf8mb4_unicode_ci
6. Klik "Create"
7. Pilih database rantaugrafi_db
8. Klik tab "Import"
9. Browse → pilih file "database.sql"
10. Klik "Go"
✅ Tunggu sampai selesai (lihat pesan sukses)

STEP 2: JALANKAN SERVER (1 menit)
----------------------------------
1. Buka XAMPP Control Panel
2. Klik "Start" untuk Apache
3. Klik "Start" untuk MySQL
4. Tunggu status jadi "Running" (hijau)

STEP 3: BUKA WEBSITE (1 menit)
-------------------------------
Buka browser dan akses:
📍 http://localhost/Rantaugrafi/

STEP 4: LOGIN ADMIN (1 menit)
------------------------------
1. Klik "Admin Panel" atau akses:
   http://localhost/Rantaugrafi/admin/dashboard.php
2. Email: admin@rantaugrafi.com
3. Password: password123
4. ⚠️ UBAH PASSWORD SETELAH LOGIN!

🐛 JIKA ADA MASALAH
===================

Error 1: "Koneksi Database Gagal"
→ Pastikan MySQL berjalan di XAMPP
→ Cek konfigurasi di includes/config.php
→ Verifikasi database sudah di-import

Error 2: "File Not Found" / 404
→ Pastikan folder Rantaugrafi ada di C:\xampp\htdocs\
→ Restart Apache
→ Cek URL sudah benar

Error 3: "Upload Gambar Tidak Berfungsi"
→ Set permission folder uploads menjadi 777
→ Cek ukuran file (max 5MB)
→ Format harus JPG/PNG/GIF

Error 4: "Lupa Password Admin"
→ Buka phpMyAdmin
→ Edit user admin di tabel users
→ Generate password baru di https://www.bcryptgenerator.com

Untuk Bantuan Lengkap:
→ Baca file DOCUMENTATION.md
→ Akses SETUP.php untuk panduan interaktif
→ Hubungi support: dev@rantaugrafi.com

💡 TIPS & TRIK
==============

1. Test Booking:
   - Daftar akun customer baru
   - Klik "Pesan Sekarang" pada paket
   - Isi form dan lanjutkan

2. Tambah Paket:
   - Login ke admin
   - Buka "Kelola Paket"
   - Klik "Tambah Paket"
   - Isi form dan simpan

3. Ubah Warna Tema:
   - Edit assets/css/style.css
   - Cari #667eea dan #764ba2
   - Ganti dengan warna pilihan Anda

4. Integrasi Payment:
   - Gunakan Midtrans, Xendit, atau platform lain
   - Modifikasi file payment.php
   - Integrasikan API key
