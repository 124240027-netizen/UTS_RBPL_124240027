🎥 SCENE 1: MEMBUKA XAMPP
═════════════════════════

[START VIDEO]

📍 FRAME 1: Desktop Windows
    Anda akan melihat desktop dengan berbagai icon

📍 FRAME 2: Cari XAMPP Control Panel

📍 FRAME 3: XAMPP Control Panel Terbuka
    Window baru muncul dengan judul "XAMPP Control Panel"

🎥 SCENE 4: IMPORT FILE database.sql
═════════════════════════════════════

[START VIDEO]

📍 FRAME 1: Klik Database rantaugrafi_db
    Di sidebar kiri, cari dan KLIK "rantaugrafi_db"
    
    Halaman akan berubah, menampilkan tabs:
    Structure | SQL | Search | Query | Export | Import

📍 FRAME 2: Klik Tab Import
    Di tabs di atas, cari tab "Import" (biasanya terakhir)
    KLIK tab tersebut
    
    Halaman akan berubah menampilkan form import

📍 FRAME 3: Form Import Muncul
    Anda akan melihat:
    
    ┌────────────────────────────────────────┐
    │ File to import:                        │
    │ [Choose File]                          │
    │                                        │
    │ Charset: [UTF-8 ▼]                    │
    │                                        │
    │ [ Continue ]  [ Cancel ]               │
    └────────────────────────────────────────┘

📍 FRAME 4: Klik "Choose File"
    Klik tombol "Choose File" atau "Browse..."
    
    File Explorer akan terbuka
    (Jendela baru untuk memilih file)

📍 FRAME 5: Navigate ke Folder Rantaugrafi
    Anda sekarang di File Explorer
    
    Lihat "Address Bar" (paling atas):
    ┌────────────────────────────────────────┐
    │ C:\Users\... atau C:\xampp\... etc     │
    └────────────────────────────────────────┘
    
    Anda perlu ke: C:\xampp\htdocs\Rantaugrafi\
    
    CARA 1: Ketik di Address Bar
    - Klik address bar
    - Hapus semua teks
    - Ketik: C:\xampp\htdocs\Rantaugrafi
    - Tekan ENTER
    
    CARA 2: Manual navigate
    - Di panel kiri, cari "This PC"
    - Double-click "Local Disk (C:)"
    - Double-click folder "xampp"
    - Double-click folder "htdocs"
    - Double-click folder "Rantaugrafi"

📍 FRAME 6: Cari File database.sql
    Di folder Rantaugrafi, cari file bernama:
    database.sql
    
    Seharusnya terlihat dengan icon dokumen/SQL

📍 FRAME 7: Klik & Open File
    DOUBLE-CLICK file "database.sql"
    
    atau
    
    SINGLE-CLICK pilih, lalu klik "Open"
    
    File Explorer akan menutup dan kembali ke phpMyAdmin

📍 FRAME 8: Kembali di phpMyAdmin
    Sekarang di form Import, Anda bisa lihat:
    - Nama file sudah tertera
    - Charset: UTF-8
    
    Tombol "Continue" atau "Go" siap diklik

📍 FRAME 9: Klik Tombol Continue/Go
    Klik tombol "Continue" atau "Go"
    
    Proses import dimulai...
    
⏳ TUNGGU 5-10 DETIK (atau lebih tergantung ukuran file)...

📍 FRAME 10: Import Selesai
    Anda akan melihat PESAN HIJAU:
    
    ✅ "Import has been successfully finished."
    ✅ "123 queries executed."
    
    atau
    
    ✅ "Import successful!"

🎥 SCENE 5: VERIFIKASI DATABASE
════════════════════════════════

[START VIDEO]

📍 FRAME 1: Lihat Sidebar Kiri
    Di sidebar kiri phpMyAdmin, lihat "rantaugrafi_db"
    
    Ada tanda ▶ atau ➕ di sampingnya (expand button)

📍 FRAME 2: Expand Database
    Klik tanda ▶ atau ➕ untuk expand
    
    Database akan membuka dan menampilkan daftar TABEL

📍 FRAME 3: Lihat Daftar Tabel
    Seharusnya ada 8 TABEL:
    
    ✅ bookings
    ✅ packages
    ✅ payments
    ✅ portfolio
    ✅ reviews
    ✅ settings
    ✅ testimonials
    ✅ users
    
    HITUNG - pastikan ada 8 (tidak kurang)

📍 FRAME 4: Klik Salah Satu Tabel (misal "users")
    KLIK tabel "users" untuk melihat struktur
    
    Halaman akan menampilkan field-field tabel:
    
    Field       | Type          | Null | Key |
    ─────────────────────────────────────────
    id          | int           | No   | PRI |
    name        | varchar(100)  | No   |     |
    email       | varchar(100)  | No   | UNI |
    password    | varchar(255)  | No   |     |
    ...dan field lainnya

📍 FRAME 5: Lihat Data di Tabel Users
    Klik tab "Browse" atau "Data"
    
    Anda akan melihat data yang ada di tabel:
    
    id | name                  | email                    | password | ...
    ──────────────────────────────────────────────────────────────
    1  | Admin Rantaugrafi     | admin@rantaugrafi.com    | [hash]   | ...
    
    Ini menunjukkan ada admin user sudah ada
